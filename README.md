#### Abstract

This research focuses on the design and implementation of a smart home automation and vehicle speed detection system using Internet of Things (IoT) technology. The proposed system enables remote monitoring and control of electrical appliances through Google Assistant, NodeMCU, Blynk, and cloud-based communication platforms.

#### Technologies Used

* Internet of Things (IoT)
* NodeMCU (ESP8266)
* Google Assistant
* Blynk Application
* IFTTT
* Wi-Fi Communication
* Embedded Systems

#### Key Features

* Smart Home Automation
* Voice-Controlled Appliance Management
* Vehicle Speed Detection
* Wireless Monitoring and Control
* Cloud-Based Communication
* Mobile Application Integration

#### Research Outcome

The proposed system successfully demonstrates low-cost home automation and remote appliance control using IoT technology while integrating vehicle speed detection and intelligent monitoring capabilities.
